# The Art of Sushi

An immersive, educational website exploring the history, types, and etiquette of sushi. Built with vanilla HTML, CSS, and JavaScript, featuring a modern dark-mode aesthetic and smooth scroll animations.

## Features

*   **Responsive Design**: Fully responsive layout that looks great on desktop, tablet, and mobile.
*   **Interactive Gallery**: Showcase of various sushi types (Nigiri, Maki, Sashimi, Temaki) with hover effects.
*   **Educational Content**: Sections dedicated to the history of sushi and dining etiquette.
*   **Custom Assets**: High-quality, AI-generated imagery.
*   **Smooth Animations**: Intersection Observer-based scroll reveals.

## Usage

Simply open `index.html` in any modern web browser to view the site.

```bash
# On macOS
open index.html
```

## Project Structure

*   `index.html`: Main HTML structure.
*   `styles.css`: All styles, including variables and responsive rules.
*   `script.js`: Interactions and scroll animations.
*   `assets/images/`: Images used in the project.
